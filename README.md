Fxex
====

**Description**

The prodigal son of an earlier repository I have on github called fxerl.
Elixir is much easier to get up and running with a working application

**Usage**

```git clone http://github.com/arthurcolle/fxex```

```cd fxex```

```mix deps.get```

```mix compile```

```iex -S mix run```

```iex(1)> Fxex.usd "USD"```

1

```iex(2)> Fxex.usd "EUR"```

0.922129

```iex(3)> Fxex.usd "GBP"```

0.663059

```iex(4)> Fxex.usd "JPY"```

119.452501
